# Lec 1 Foundation

Slidev slides for a C language embedded development course.

## Development

```bash
pnpm install
pnpm run dev
```

The local Slidev server opens at <http://localhost:3030>.

## Build

```bash
pnpm run build
```

## Export

```bash
pnpm run export
```

PDF export may require Playwright's Chromium browser.
